This project is a personal portfolio and blog website created to showcase my technical skills, share insights, and engage with the tech community.

## Key Features

- **Static Site Generation:** Built with Hugo, a fast and flexible static site generator, ensuring high performance and security.
- **Responsive Design:** The website is fully responsive, providing an optimal viewing experience across a wide range of devices. It was built using HTML, CSS, Bootstrap, and JavaScript.
- **Integrated Blog:** The site includes a blog where I share my insights and experiences in computer science, with a focus on machine learning, data analytics, and innovation.
- **Continuous Deployment:** Hosted on Netlify, the website is automatically deployed from the GitHub repository, enabling a seamless CI/CD workflow.

## Project Goals

The primary goal of this project was to create a platform to not only showcase my work but also to share my passion for technology and learning with a broader audience.
