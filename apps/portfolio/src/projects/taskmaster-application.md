TaskMaster is a clean and simple Java-based application engineered for efficient task management.

## Key Features

- **Intuitive UI:** The application is designed with a user-friendly interface to make task management as simple as possible.
- **Task Sorting:** It includes a custom algorithm to sort tasks, helping users prioritize and organize their to-do lists.
- **Advanced Search:** A search function that utilizes regular expressions allows for powerful and flexible querying of tasks.
- **Robust Error Handling:** The application is designed with strong error handling mechanisms to ensure a stable user experience.
- **Cloud Integration:** The project is currently being migrated to AWS to provide secure data storage and user authentication, making it a scalable, cloud-native application.

## Technical Implementation

The core of the application is built with Java. The ongoing migration to AWS demonstrates skills in cloud services, including database management and authentication services.
